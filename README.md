# Unreal Engine 2D Platformer Game

A complete 2D platformer game developed using Unreal engine & blueprints.

## Features
- Player movement using WASD /arrow keys / spacebar
- Turret enemies
- static spikes & retractable spikes
- falling platforms
- Timer based gameplay
- UI screen with restart button

## Gameplay systems
- collision detection
- game loop handling
- UI & restart logic

## Status
Completed basic gameplay systems. Open to further polish & improvements.

## Learning source
Built by following and adapting multiple beginner Unreal Engine tutorials, 
with custom modifications to understand core gameplay mechanics.

## Controls
WASD / arrow keys –move  
restart button – restart level
